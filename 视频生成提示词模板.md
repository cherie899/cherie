# 视频生成提示词模板

> 适用工具：即梦 / Kling / 可灵 / Pika / Runway 等主流视频生成工具
> 使用方法：复制下方模板，替换 `【】` 中的内容即可

---

## 完整模板

```
【任务类型】视频片段创作
【生成比例】16:9
【生成时长】X秒

【生成需求】
0-X秒：【镜头运动 + 空间建立】描述镜头从哪里来、到哪里去、最终定格在什么画面，营造什么情绪氛围。
X-X秒：【核心事件】描述主要动作/特效/冲突/转折，注意写清楚视觉特效细节（光影、粒子、裂痕、能量等）。
X-X秒：【角色聚焦 + 情绪高潮】描述镜头推近至角色特写，角色动作、表情、周围环境响应，画面最终定格姿态。

【角色描述】
角色名（参考图X）：性别、年龄段、发型发色、服饰风格、核心道具/武器、标志性特征。

【对白/音频】
对白（无字幕，角色配音）：「具体台词内容。」
语气：XX、XX、XX（如冷酷、威严、轻蔑）
环境音效：XX（如雷电轰鸣、冰面碎裂、风声呼啸）
BGM风格：XX（如史诗交响、电子低音、古风弦乐）

【视觉风格】
整体保持XX风格，XX与XX主色调，背景为XX，画面充满XX的情绪/力量感。

【质量约束】
五官清晰不变形、动作自然流畅、4K超高清、画面稳定。
```

## 六层结构速查表

| 层级 | 作用 | 示例关键词 |
|------|------|-----------|
| **一、元信息** | 告诉引擎基础参数 | 任务类型、比例、时长 |
| **二、时间轴** | 按秒拆分动作（三幕式） | 建立空间→事件爆发→角色聚焦 |
| **三、角色参考** | 锚定人物外观 | 性别、服饰、武器、参考图 |
| **四、台词音频** | 控制声音表演 | 具体台词、语气、音效、BGM |
| **五、视觉风格** | 全局氛围底色 | 风格、色调、背景、情绪感 |
| **六、质量兜底** | 防止AI崩坏 | 五官清晰、动作流畅、4K、稳定 |

## 常用镜头运动词汇库

| 运动类型 | 中文描述 | 适用场景 |
|---------|---------|---------|
| 推镜头 | 缓慢推近 / 极速推近 / 变速推近 | 聚焦角色、营造压迫感 |
| 拉镜头 | 缓慢后拉 / 快速后拉 / 跟随后拉 | 揭示全貌、惊喜感 |
| 摇镜头 | 水平摇摄 / 垂直摇摄 / 环绕摇摄 | 扫描环境、建立关联 |
| 移镜头 | 横向平移 / 跟随移动 / 环绕移动 | 伴随角色、展开空间 |
| 俯冲 | 从高空极速坠落 | 冲击感、开场震撼 |
| 升起 | 从地面缓慢升起 | 宏大感、史诗开场 |
| 环绕 | 围绕角色360度旋转 | 全方位展示、仪式感 |
| 手持 | 轻微晃动 / 剧烈抖动 | 真实感、紧张混乱 |
